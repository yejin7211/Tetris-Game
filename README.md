# Tetris-Game
 첫 자바스크립트 공부 이후 배운걸 활용하고자 테트리스 게임을 만들었습니다.

--------------------------------------------------------------------
프로젝트 설명
 여러 가지 테트리스 조각들이 게임창 위에서 떨어집니다. 사용자는 테트리스 조각들을 방향키를 이용하여 적절하게 조각을 회전시킬 수 있습니다. 이 때, 바닥에 떨어진 테트리스가 모두 한 줄로 채워졌을 때, 해당 행을 없앱니다. 만약, 테트리스가 게임창의 맨 위에 닿으면 게임은 종료됩니다.
